Intro

Angular basics and setup overview

What is Angular?

  • Angular is a frontend JavaScript framework
  • It helps with building interactive, modern web user interfaces
  • It's also a collection of tools & features like: CLI, Debugging Tools, IDE Plugins

Why using Angular?

  • Simplifies building complex, interactive web UI
  • Write declarative code instead of imperative code
  • Separation of concerns via components
  • Uses object-oriented programming concepts & principles
  • Uses TypeScript ✅

New Project

  • Angular Installation: npm install -g @angular/cli
  • New Project: ng new project-name
  • Extensions: Angular Language Service (angular.io), Angular Essentials
  • Run Project: npm start or ng serve
  • Local Website: http://localhost:4200/
  • Stop Project: ctrl + c

Angular Project Structure

Angular project contains TypeScript files, JSON files, src folder and other important config files.

Important files/folders

  • TypeScript Files → compiled to JavaScript (handled by Angular CLI)
  • JSON Files
    • package.json → manages dependencies
    • angular.json → Angular CLI configuration
  • src/
    • app/ → components & services ✅ (main development)
    • styles.scss → global stylesheet
    • index.html → main HTML file
    • main.ts → entry point
    • assets/ → static files
  • Other Important Files
    • .editorconfig → code formatting rules
    • .gitignore → git tracking rules

✅ Focus mainly on the app/ folder while developing.

Build Project

ng build --output-path=docs --base-href /templates/
